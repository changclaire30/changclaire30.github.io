Hexo安裝主題時錯誤訊息

6月 10 2021

第一次使用Hexo部署並更換主題時(我使用的是Polarbear)，主要遇到兩個在教學步驟中沒有出現的問題：

  1. Hexo下載主題到資料夾，並測試server時，在localhost出現：

{% extends ‘_layout.swig‘ %} {% import ‘_macro/post.swig‘ as post_template %}

解法是自行手動安裝swig
npm i hexo-renderer-swig

  1. 使用Hexo部署到git時出現Error訊息：

ERROR Deployer not found: git

解法也是手動安裝deployer
npm install hexo-deployer-git --save
