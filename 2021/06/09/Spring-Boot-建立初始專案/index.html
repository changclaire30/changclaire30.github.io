Spring Boot 建立初始專案

6月 09 2021

Spring Boot是Java程式語言的網頁後端Framework，在其官方網站有簡單的設置教學(英文)，是使用 start.spring.io 這個圖形介面化網站選擇後，點選”Generate”下載JAR檔開啟，也可以將同樣設定分享(Share)給其他人，十分簡單明瞭。

如圖中所示，有不同的選項包括Java的版本，根據所選的Java版本，可在Oracle網站下載對應的JDK。

在右方欄位可根據自己的需求選擇dependencies，例如Spring Web、Spring Data JPA、不同的database等等。

下載並開啟

下載完成之後會是zip檔案，解壓縮之後到開發環境IDE開啟，我是使用IntelliJ(有免費的Community版本)，但也有人會使用Visual Studio Code，例如這篇痞客邦的鐵人文章。

在prom.xml檔案中可看到專案的詳細內容包括dependencies，若對該檔案有更動，建議在檔案名稱點選右鍵 > Maven > Reload project。

專案跑起來

而在src/main/java/com/example/demo這個檔案夾底下，可以找到程式檔案DemoApplication.java，找到畫面中的綠色箭頭，選擇Run ‘DemoApplication’就可以跑專案了，或也可以在Terminal下指令，可參考上面設置教學(英文)的Step 3。

等一下下之後，下方會跳出視窗，如果沒有紅色的error訊息，而是白色的Spring圖形就是成功了。若是選擇網站開發，也可以在DemoApplication的檔案裡面撰寫不同的return值，在http://localhost:8080/ 查看結果。
